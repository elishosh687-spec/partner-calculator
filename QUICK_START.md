# ⚡ מדריך התחלה מהיר

## 🎯 3 צעדים להפעלת המערכת

### צעד 1️⃣: העלאת חוקי אבטחה ל-Firebase (קריטי!)

#### אופציה א' - אוטומטי 🚀 (מומלץ!)
```bash
# התקנה חד-פעמית
npm install -g firebase-tools
firebase login

# העלאה אוטומטית
npm run deploy:rules
npm run deploy:indexes
```
⏱️ זמן: 1 דקה  
📖 מדריך מלא: `FIREBASE_CLI_SETUP.md`

#### אופציה ב' - ידני
```
1. פתח: https://console.firebase.google.com
2. בחר פרויקט: partner-calcilator
3. לך ל: Firestore Database → Rules
4. העתק את התוכן מקובץ: firestore.rules
5. הדבק והקלק: Publish
```
⏱️ זמן: 2 דקות

---

### צעד 2️⃣: יצירת Index ב-Firestore

**אופציה א' - אוטומטי (מומלץ):**
```
1. הרץ: npm run dev
2. נסה להתחבר כשותף
3. אם יש שגיאה - לחץ על הלינק שFirebase שולח
4. המתן 2-3 דקות
```

**אופציה ב' - ידני:**
```
Firebase Console → Firestore Database → Indexes → Create Index
Collection: transactions
Fields:
  - partnerId (Ascending)
  - createdAt (Descending)
Query scope: Collection
```

⏱️ זמן: 3 דקות

---

### צעד 3️⃣: יצירת משתמש ראשון (הבוס)

```
1. הרץ: npm run dev
2. לחץ: "אין לך חשבון? הירשם"
3. מלא:
   ✏️ שם: [השם שלך]
   👑 תפקיד: בוס
   📧 אימייל: boss@example.com
   🔒 סיסמה: [לפחות 6 תווים]
4. לחץ: "צור חשבון"
```

⏱️ זמן: 1 דקה
